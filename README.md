# 🚀 Fake News Detection App - Streamlit Frontend

A modern, interactive web application for detecting fake news using Machine Learning.

## 📋 Features

- **Real-time Detection**: Instantly analyze news articles
- **Confidence Scores**: Get detailed probability predictions
- **Beautiful UI**: Modern, responsive interface built with Streamlit
- **Text Cleaning**: Automatic preprocessing of input text
- **Detailed Analytics**: View word counts, character counts, and more
- **Mobile Friendly**: Works on desktop, tablet, and mobile devices

## 🛠️ Setup Instructions

### Step 1: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 2: Train the Model
First, ensure your CSV dataset is in the workspace directory (fake_or_real_news.csv):
```bash
python train_model.py
```

This will create two model files:
- `fake_news_model.pkl` - The trained Naive Bayes classifier
- `vectorizer.pkl` - The TF-IDF vectorizer

### Step 3: Run the Streamlit App
```bash
streamlit run streamlit_app.py
```

The app will open in your browser at `http://localhost:8501`

## 📊 How It Works

1. **Input**: Paste or type a news article in the text area
2. **Processing**: The text is cleaned and vectorized using TF-IDF
3. **Prediction**: Naive Bayes classifier predicts if it's fake or real
4. **Results**: View confidence scores and detailed analysis

## 🎯 Model Details

- **Algorithm**: Multinomial Naive Bayes
- **Feature Extraction**: TF-IDF Vectorizer
- **Training Data**: Fake and Real news articles
- **Preprocessing**: 
  - Lowercase conversion
  - Punctuation removal
  - Number removal
  - Whitespace normalization

## 🚀 Deployment

To deploy this app:

### Streamlit Cloud
1. Push your code to GitHub
2. Connect your repository to Streamlit Cloud
3. Deploy with one click

### Docker
```dockerfile
FROM python:3.9
WORKDIR /app
COPY . .
RUN pip install -r requirements.txt
CMD ["streamlit", "run", "streamlit_app.py"]
```

### Local Server
```bash
streamlit run streamlit_app.py --server.port 8501 --server.address 0.0.0.0
```
